computedFrom

computedFrom is a helper function that combines the values of Observables or Signals and emits the combined value. It also gives us the possibility to change the combined value before emitting it using rxjs operators.

It is similar to combineLatest, but it also takes Signals into consideration.

Usage

computedFrom accepts an array or object of Observables or Signals and returns a Signal that emits the combined value of the Observables or Signals. By default, it needs to be called in an injection context, but it can also be called outside of it by passing the Injector in the third argument options object. If your Observable doesn‚Äôt emit synchronously, you can use the startWith operator to change the starting value, or pass an initialValue in the third argument options object.

It can be used in multiple ways:

  1. Combine multiple Signals
  2. Combine multiple Observables
  3. Combine multiple Signals and Observables
  4. Using initialValue param
  5. Use it outside of an injection context
